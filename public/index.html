About This Project

Full-stack AI code reviewer with DevOps pipeline.

// What it does

Analyzes any code using Groq AI — detects bugs, calculates time/space complexity, shows corrected code, and provides an optimized version. All saved to MongoDB.

// Tech stack

HTML5CSS3 JavaScriptNode.js ExpressMongoDB MongooseGroq AI DockerGitHub Actions Vercel

// DevOps

Full CI/CD pipeline: GitHub Actions (lint → security → Docker build → Vercel deploy → Slack notify). Multi-stage Docker with Trivy CVE scanning. View DevOps Dashboard →

// Architecture

Frontend → Groq API for AI review → Express backend → MongoDB Atlas. Deployed on Vercel (serverless functions) + Docker for self-hosting.
