// Quick Start Guide
01
🐳
Run with Docker
Spin up the full stack locally — app + MongoDB — with one command.
cp .env.example .env # edit .env with your keys docker compose up -d # → http://localhost:3000
02
⚙️
Deploy to Vercel
Push to GitHub and let GitHub Actions handle the rest automatically.
npm i -g vercel vercel login vercel link # add secrets in GitHub repo git push origin main
03
🔐
Add GitHub Secrets
Go to Settings → Secrets → Actions and add the 4 required secrets above.
VERCEL_TOKEN=... VERCEL_ORG_ID=... VERCEL_PROJECT_ID=... # Set MONGO_URI in Vercel # dashboard env vars
04
🔬
Open a Pull Request
Every PR gets a live preview URL posted as a comment automatically.
git checkout -b feature/xyz # make changes git push origin feature/xyz # → open PR on GitHub # → bot posts preview URL
